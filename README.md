# probot-carbon-copy-content

> A GitHub App built with [Probot](https://github.com/probot/probot) that Edit files in one repo, have them seamlessly copied to other repos as pull requests.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
